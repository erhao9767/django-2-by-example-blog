## 运行环境

| Project | Status | Description |
|---------|--------|-------------|
| python          | 3.6.5	| 在这个版本以及以上都可以 |
| django          | 2.1.2   | 在这个版本以及以上都可以 |
| django-taggit   | 0.23.0  | 必须此版本 |

## 安装环境
`pip3 install django`

`pip3 install django-taggit 0.23.0`

使用Pycharm直接运行即可，
或者使用命令
`python manage.py runserver`
